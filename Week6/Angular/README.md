# Week 6 – Angular Hands-ons

## Overview

This folder contains the Angular hands-on project completed as part of **Week 6** of the Cognizant Digital Nurture 5.0 (Java FSE) program.

The project focuses on building a modern single-page web application using **Angular**. It demonstrates component-based development, routing, data binding, services, and responsive user interface design.

---

## Project Completed

### Student Course Portal

Developed a Student Course Portal application that allows users to manage and view course-related information through an intuitive Angular interface.
