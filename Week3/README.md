# Week 3 – Spring REST using Spring Boot 3

## Overview

This week focuses on building RESTful web services using **Spring Boot 3**. The hands-ons cover the fundamentals of REST APIs, request handling, JSON responses, and JWT-based authentication. Through these exercises, I learned how to design, develop, and test RESTful services using Spring Boot.

The projects include creating REST endpoints, implementing country-related APIs, and securing services using JSON Web Tokens (JWT).

---

## Hands-ons Completed

### Spring REST

- Created a Spring Boot REST project using Maven.
- Implemented a Hello World REST API.
- Developed REST APIs for Country services.
- Implemented API to retrieve country details using country code.
- Configured REST controllers and request mappings.

### JWT Authentication

- Created a JWT Authentication Service.
- Generated JWT tokens for authenticated users.
- Implemented secure authentication flow using Spring Security and JWT.
