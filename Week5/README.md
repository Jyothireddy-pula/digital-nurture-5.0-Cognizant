# Week 5 – React Hands-ons

## Overview

This week focuses on building interactive web applications using **React.js**. The hands-ons cover the fundamentals of React development, including component-based architecture, state management, event handling, forms, routing, and application development through multiple mini projects.

The exercises helped strengthen my understanding of modern frontend development and how React simplifies building reusable and maintainable user interfaces.
